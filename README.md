# ⚙️ نصب

1. آپلود فایل‌ها

فایل‌های پروژه را روی هاست PHP آپلود کنید.

2. ساخت ربات

از BotFather یک ربات ساخته و توکن دریافت کنید.

3. اجرای نصب

آدرس زیر را باز کنید:

https://YOUR-DOMAIN.com/SasanianEmpire/installer.php

سپس اطلاعات زیر را وارد کنید:

- Bot Token
- Admin ID
- Bot Username
- Domain URL

4. اتمام نصب

پس از نصب، ربات آماده استفاده خواهد بود.
